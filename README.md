# Yellowstone Common

[![Go](https://github.com/evrblk/yellowstone-common/actions/workflows/go.yml/badge.svg)](https://github.com/evrblk/yellowstone-common/actions/workflows/go.yml)
[![Go Report Card](https://goreportcard.com/badge/github.com/evrblk/yellowstone-common)](https://goreportcard.com/report/github.com/evrblk/yellowstone-common)

This repository contains some reusable utils for Everblack services.

* `cache` - simple caching with TTL
* `encoding/base62` - Base62 implementation
* `metrics` - handy Prometheus metrics server wrapper
* `workers` - interval worker implementation
* `utils` - missing functions from `samber/lo`,
